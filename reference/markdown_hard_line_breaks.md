# hard line breaks

Custom rmarkdown template based on the standard
[`rmarkdown::html_document()`](https://pkgs.rstudio.com/rmarkdown/reference/html_document.html),
but with hard line breaks. Will add the pandoc '+hard_line_breaks'
argument if the origin format is markdown.

## Usage

``` r
markdown_hard_line_breaks(...)
```

## Arguments

- ...:

  all other arguments passed to
  [`rmarkdown::html_document()`](https://pkgs.rstudio.com/rmarkdown/reference/html_document.html)
